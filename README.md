# 🏧 Python ATM — Web Edition

A colorful, browser-based ATM interface built with plain HTML, CSS, and JavaScript.

**🔗 Live Demo:** https://ramprasanth07.github.io/python-atm/

## Features

- 🔐 4-digit PIN login (3 attempts, then account blocks)
- 💰 Check balance
- 💵 Withdraw money (multiples of ₹100 only, checked against balance)
- 🐷 Deposit money
- 🧾 Mini statement (transaction history)
- 🔑 Change PIN
- 🖨️ Printed-style receipt after each transaction

## Files

| File | Description |
|---|---|
| `index.html` | Colorful browser-based ATM interface, runs entirely client-side |

## Running

Just open `index.html` in any browser — no installation needed. Default PIN: `2007` · Starting balance: ₹5000. Data is kept in memory only and resets on page reload.

## Tech

- Plain HTML, CSS, and JavaScript (no frameworks, no build step)
